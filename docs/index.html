Getting Started

Installation

Install Unified Theme via npm or include it directly in your HTML:

Via NPM

npm install unified-theme

Via CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/unified-theme/dist/unified-theme.css"> <script src="https://cdn.jsdelivr.net/npm/unified-theme/dist/unified-theme.js"></script>

Basic Usage

<div class="container"> <h1 class="text-heading-xl text-primary-600">Hello World</h1> <p class="text-body-lg text-neutral-700">Welcome to Unified Theme!</p> <button class="btn btn-primary btn-lg">Get Started</button> </div>
